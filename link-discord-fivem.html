Liaison entre Discord et FiveM

Découvrez comment lier votre compte Discord à FiveM.

Démarrer le tutoriel Revenir au centre d’aide

Avant-propos

Il arrive que l’accès à un serveur FiveM vous soit refusé. Cela peut être dû au fait que votre compte Discord n’est pas lié à FiveM, ou que vous ne faites pas partie du serveur Discord du serveur FiveM, car certains serveurs l’exigent. Suivez les étapes ci-après pour régler le soucis.

1. Autoriser l’application FiveM

Il est essentiel d’ajouter FiveM dans vos applications autorisées de Discord

Pour ajouter FiveM dans vos applications autorisées :

  1. Fermez totalement FiveM et ses processus en arrière-plan.
  2. Ouvrez l’application Discord et allez dans « Paramètres utilisateur ».
  3. Ouvrez l’onglet « Applications autorisées ».
  4. Si vous trouvez FiveM, cliquez sur « Retirer l’autorisation ».

2. Relancer les applications

Discord ne doit pas s’exécuter en mode « Administrateur ».

Assurez-vous de lancer les applications dans cette ordre :

  1. Discord.
  2. FiveM.

Suite à cela, il est possible d’avoir une pop-up sur l’application Discord vous demandant d’autoriser la connexion entre FiveM et Discord. Si c’est le cas, autorisez l’accès.
